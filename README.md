# ML & FoAI demos

This repository includes (unfinished) demos based on the TUM lectures IN2406 and IN2064.

__WARNING:__ This project is neither __done__ nor __optimized__. It should run reasonably well on Chromium-based browsers, no guarantees are made for Firefox.

## Developing

Once you've created a project and installed dependencies with `npm install` (or `pnpm install` or `yarn`), start a development server:

```sh
npm run dev
```
